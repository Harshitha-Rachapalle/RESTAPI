## SOAP API (Old but Still Asked)

SOAP (Simple Object Access Protocol) is a strict, **XML-based messaging protocol** used for communication between applications.

**Real‚Äëtime example:** Banking systems

Example scenario:

‚ùå Cons

## REST API 

REST (Representational State Transfer) is an architectural style that **uses HTTP methods to interact with resources.**

REST APIs are **lightweight, fast, and scalable.**

|Method    | Meaning  |
|--------- |  ----------|
|`GET `    |Retrieve data|
|`POST`    |Create data   |
|`PUT`     |Update entire resource |
| `PATCH`   |Partial update    |
|`DELETE`  |Delete resource |

## GraphQL API

GraphQL allows clients to request only the data they need, nothing more, nothing less.

GraphQL is a **query language for APIs** developed by Facebook.

The client decides exactly what data it needs.

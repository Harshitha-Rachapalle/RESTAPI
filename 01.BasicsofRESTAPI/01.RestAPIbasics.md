### What is Statelessness?

Statelessness means **each API request is independent**.

The server:

Does not store client state
Does not remember previous requests

Every request must contain all information needed to process it.

**State** is **stored** on the **client**, not on server.

### What If API Was Stateful?

If stateful:

Server stores session

Remembers logged-in user

Needs session cleanup

Harder to scale

This is how old web applications worked.
